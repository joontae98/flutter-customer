# Customer_list

Customer의 이름과 지불할 가격을 등록해 list를 생성하여 Customer가 금액을 지불하였는지 확인하는 app

## How to use?
1. +버튼의 Add Customer를 이용하여 Customer의 정보를 등록
2. Customer가 지불하여야하는 금액을 전부 지불한다면 Check Box를 체크
3. 지불 완료된 고객을 지우고 싶으면 +버튼의 Done Delete
